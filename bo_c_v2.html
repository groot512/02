주요 개선 기능
더 안전하고, 더 편리하게
2단계 사업을 통해 보훈마켓이 이렇게 달라집니다.

서버 이중화 및 안정성 강화

로드밸런서 기반 서버 4대 구성으로 장애 시에도 서비스가 중단되지 않는 환경을 구축합니다.

인프라

개인정보 보호 강화

AES-256 암호화 적용, 관리자 보안 정책 강화로 회원 개인정보를 안전하게 보호합니다.

보안

OCR 자동 회원가입

증빙자료 촬영만으로 정보를 자동 인식합니다. 복잡한 수기 입력이 필요 없습니다.

편의성

스마트 혜택지도

카테고리 필터, 검색, 퀵 지역 이동 등 강력해진 지도로 혜택처를 빠르게 찾습니다.

지도

실시간 알림 시스템

할인 적용, 새로운 혜택, 이벤트 소식을 앱 푸시와 알림톡으로 받아보실 수 있습니다.

알림

프랜차이즈 확장 대응

범용 API 구조로 다양한 프랜차이즈와 빠르게 연동, 혜택처가 계속 확대됩니다.

확장
이용방법
혜택, 이렇게 받으세요!
간편한 4단계로 보훈 혜택을 누리실 수 있습니다.
1

앱 설치 및 회원가입

OCR 자동 인식으로 증빙자료를 촬영하면 빠르게 가입이 완료됩니다.

2

대상자 인증 및 승인

보훈 대상자 확인 후 자동 승인이 처리됩니다.

3

혜택처 방문

스마트 지도에서 가까운 가맹점을 검색하고 방문하세요.

4

할인 혜택 적용

회원증 제시 또는 바코드 스캔으로 즉시 할인이 적용됩니다.
